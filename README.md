## Getting Started
### Installation

Choose your preferred package manager:

```bash
# npm
npm install

# yarn
yarn

# pnpm
pnpm install

# bun
bun install
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
